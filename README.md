# Data Insights & Business Solutions Portfolio
This portfolio showcases a collection of my data analysis projects, which I completed for academic purposes, self-learning, and personal enjoyment. The projects are presented in various formats, including Excel files, Power BI files, Jupyter notebooks, and R Markdown files.

- ## Power BI

- [Power BI Health Analytics](https://github.com/SudheerKumarTatavalu/Data-analysis-projects/tree/e5064a6c2f443738a5d3b9c2d44ec22eb8219637/Power%20BI%20Health%20Analytics) This project delivers a comprehensive analysis of the U.S. healthcare landscape from 2019 to 2020, utilizing Power BI for interactive visualizations and DAX calculations to derive key metrics. It focuses on hospital performance, patient outcomes, and healthcare provider relationships, providing actionable insights for informed decision-making. The project emphasizes data cleaning and transformation, ensuring accuracy while offering valuable resources for stakeholders to enhance care quality and operational efficiency.

- [Personal Finance Dashboard in Power BI](https://github.com/SudheerKumarTatavalu/Data-analysis-projects/tree/e5064a6c2f443738a5d3b9c2d44ec22eb8219637/Personal%20Finance%20Dashboard%20in%20Power%20BI) This project develops a personal finance dashboard in Power BI, allowing users to monitor expenses, income, and savings while setting and tracking financial goals. Interactive visualizations enhance user engagement and facilitate easy exploration of financial data.

- ## QuickSight

- [YouTube_Data_Analysis](https://github.com/SudheerKumarTatavalu/Data-analysis-projects/tree/e5064a6c2f443738a5d3b9c2d44ec22eb8219637/YouTube_Data_Analysis) This project focuses on managing and analyzing structured and semi-structured YouTube video data through an ETL system, leveraging AWS services for scalability and cloud processing. Utilizing Amazon QuickSight, we build an interactive dashboard to visualize video performance trends and insights. The project aims to streamline data ingestion and storage while ensuring real-time analytics capabilities for informed decision-making.

- ## Python

- - [E-Comm Sales Insights](https://github.com/SudheerKumarTatavalu/Data-analysis-projects/tree/e5064a6c2f443738a5d3b9c2d44ec22eb8219637/E-Comm%20Sales%20Insights) This project analyzes Kmart's 2019 sales data to inform its 2020 sales strategy, providing insights into the best month for sales, top-selling products, and optimal advertising times. The analysis utilizes Python, Pandas, and data visualization libraries to extract key findings and recommendations for improving sales performance.
